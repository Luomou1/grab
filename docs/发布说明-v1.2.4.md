# v1.2.4 发布说明

`v1.2.4` 增加在线更新缓存清理，避免更新安装包长期留在本机临时目录。

## 变更

- 程序启动时会清理 `%TEMP%\grab-updates` 下的旧版本安装包缓存。
- 残留的 `.part` 下载临时文件会自动删除。
- 当前版本缓存目录会保留，避免误删刚完成更新流程使用的安装包目录。

## 验证

- `python -m compileall grab_app run_app.py`
- `python -m pytest tests`
- `.\scripts\build_installer.ps1`

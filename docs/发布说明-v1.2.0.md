# v1.2.0 发布说明

`v1.2.0` 将程序可见名称统一为 `grab`，并把更新功能升级为应用内在线更新。

## 新增与变更

- 程序窗口名、安装目录、开始菜单入口、桌面快捷方式和安装包名称统一改为 `grab`。
- 顶部工具栏“检查更新”不再跳转网页：
  - 自动读取 GitHub 最新 Release。
  - 点击“立即更新”后在应用内下载安装包。
  - 下载完成后校验 GitHub Release 提供的 SHA256。
  - 校验通过后启动安装器并退出当前程序。
- 更新弹窗可查看 Release 更新内容。

## 验证

- `python -m compileall grab_app`
- `python -m pytest tests`
- `.\scripts\build_installer.ps1`

## 安装包

从 GitHub Release 下载：

```text
grab_Setup_v1.2.0.exe
SHA256: ABFAFFEBF7BCC49B7C49215F57666D858FF21EA85EC31B7DFD934844487D96EA
```

说明：已安装 `v1.1.0` 的用户仍需手动下载安装这一次版本；安装 `v1.2.0` 后，后续版本即可走应用内在线更新。

# v1.2.2 发布说明

`v1.2.2` 优化了应用内在线更新进度弹窗的视觉效果。

## 变更

- 用项目内自定义深色弹窗替换系统默认 `QProgressDialog`。
- 更新下载进度文字改为高对比颜色，解决浅色文字看不清的问题。
- 进度条样式统一到主界面的深色背景与青色进度色。
- 下载百分比使用固定数字区域，避免进度刷新时界面跳动。

## 验证

- `python -m compileall grab_app`
- `python -m pytest tests`
- `.\scripts\build_installer.ps1`

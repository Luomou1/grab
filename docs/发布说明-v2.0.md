# v2.0 发布说明

`v2.0` 是一次大版本更新，重点加入 XY 位移台与空间扫描能力，把原有相机 / PZT 采集流程扩展为可做二维区域规划、位置配准、扫描执行和结果保存的采集系统。

## 主要变更

- 新增 XY 位移台控制模块，包含设备连接、坐标模型、安全边界、轨迹执行、手动移动和高级运动控制。
- 新增空间扫描模块，支持扫描区域规划、视场覆盖计算、位置配准、图像拼接数据组织和扫描结果落盘。
- 主界面增加 XY 位移台控制、空间扫描参数、区域规划预览、扫描任务进度与设备状态显示。
- 新增相机健康状态检查，便于在扫描前确认相机连接和采集状态。
- 打包流程加入 XY 位移台 SDK DLL 和设备配置文件，安装包可随程序分发位移台运行依赖。
- 补充 XY 位移台、空间规划、空间扫描 Worker、地图显示、UI 集成和相机健康检查等自动化测试。

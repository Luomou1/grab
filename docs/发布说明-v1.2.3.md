# v1.2.3 发布说明

`v1.2.3` 更新了程序图标，使桌面快捷方式、安装包和 exe 图标更符合采集程序定位。

## 变更

- 新增相机镜头 + 扫描线风格的 `grab` 图标。
- PyInstaller 生成的 `grab.exe` 使用新图标。
- Inno Setup 安装包使用新图标。
- 桌面快捷方式和开始菜单入口会继承 `grab.exe` 图标。

## 验证

- `python -m compileall grab_app`
- `python -m pytest tests`
- `.\scripts\build_installer.ps1`

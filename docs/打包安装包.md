# 打包安装包说明

本项目可以打包为 Windows 安装包，目标产物为：

```text
release\HTGE采集程序_Setup_v1.0.0.exe
```

安装后会创建程序目录、开始菜单快捷方式、卸载入口，并可选创建桌面快捷方式。

普通用户无需自行打包，可直接从 [GitHub Releases](https://github.com/Luomou1/grab/releases/latest) 下载安装包。

## 前置条件

- Windows x64
- Python 3.11+
- 项目依赖见 `requirements.txt`
- PyInstaller
- Inno Setup 6
- 华腾相机 SDK / 驱动

安装 Python 依赖和 PyInstaller：

```powershell
.\scripts\build_installer.ps1 -InstallDeps
```

如果只想先生成免安装目录，不生成安装包：

```powershell
.\scripts\build_installer.ps1 -SkipInstaller
```

完整生成安装包：

```powershell
.\scripts\build_installer.ps1
```

## 华腾 SDK 处理

默认脚本会尝试把以下目录中的 x64 SDK DLL 打进程序包：

```text
D:\HuaTengVision\SDK\X64
```

这些 DLL 会放到打包后的 `grab_app\camera` 目录，程序启动时会优先把该目录加入 DLL 搜索路径。

如果不希望随安装包分发 SDK DLL，可以使用：

```powershell
.\scripts\build_installer.ps1 -NoVendorSdk
```

这种情况下，目标电脑必须已经安装华腾官方程序包，并能从默认路径加载 SDK：

```text
D:\HuaTengVision\SDK\X64
D:\HuaTengVision\SDK
```

注意：相机驱动是否允许随应用安装包分发，需要以华腾 SDK 授权条款为准。即使 DLL 已随程序打包，目标电脑仍可能需要安装相机 USB/GigE 驱动。

## 发布安装包

本地 `release/` 目录仅保存构建产物，已加入 `.gitignore`，安装包不应直接提交到 Git 历史。正式版本通过 GitHub Release 附件分发。

发布前至少确认：

- `python -m pytest` 全部通过
- 安装包可正常静默安装和卸载
- 已安装程序可启动并进入 GUI 事件循环
- 打包目录中的华腾 SDK DLL 能被加载
- Release 页面记录安装包 SHA256

计算 SHA256：

```powershell
Get-FileHash -Algorithm SHA256 .\release\HTGE采集程序_Setup_v1.0.0.exe
```

## 常见问题

### 安装后提示找不到相机

优先检查：

- 目标电脑是否安装华腾相机驱动
- 相机是否能在华腾官方软件中枚举
- 防火墙 / 网卡配置是否影响 GigE 相机
- 安装包是否包含 SDK DLL，或目标电脑是否有 `D:\HuaTengVision\SDK\X64`

### 安装后程序打不开

先运行免安装目录中的 exe，或在 PowerShell 中启动 exe，查看是否有缺失 DLL 的错误。

### 是否推荐 onefile

不推荐第一版使用 `onefile`。本程序依赖 PySide6、OpenCV、numpy 和相机 SDK DLL，`onedir` 更容易定位缺失 DLL 和驱动问题。

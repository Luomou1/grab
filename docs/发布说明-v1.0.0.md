# v1.0.0 发布说明

`v1.0.0` 是 HTGE 相机 + PZT 采集程序的首个 Windows 安装版。

## 主要功能

- 华腾相机连接、连续预览、软触发和抓拍
- `8bit` 与 `12bit packed` 图像采集和 TIFF 保存
- 传感器 ROI 鼠标框选、静态预览、坐标同步及恢复全幅
- PZT 串口 / UDP 连接、手动位移和扫描控制
- 普通扫描与中心扫描
- 动态曝光 / 增益范围、扫描参数锁定和帧统计
- 暗场 / 平场采集与校正
- 异步保存队列、扫描元数据、CSV 和 summary 输出

## 安装要求

- Windows x64
- 与相机型号匹配的华腾相机驱动
- GigE 相机需要正确配置网卡、IP 和防火墙
- 使用 PZT 时需要可用的串口或 UDP 网络连接

安装包已包含 Python、PySide6、OpenCV、NumPy 和华腾 SDK 核心 DLL，无需另行安装 Python。

## 验证结果

- 自动化测试：15 项全部通过
- PyInstaller Windows x64 构建通过
- Inno Setup 安装包编译通过
- 安装、程序启动和卸载冒烟测试通过
- 安装包内华腾 SDK DLL 加载测试通过

## 已知限制

- 相机和 PZT 的最终兼容性取决于目标电脑驱动、网卡和设备固件。
- 未连接硬件时，相机或 PZT 连接失败属于预期现象。
- 安装包未进行商业代码签名，Windows 可能显示未知发布者提示。
- 华腾 SDK DLL 和驱动的再分发应遵守厂商授权条款。

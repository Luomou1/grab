# v1.1.0 发布说明

`v1.1.0` 更新了扫描输出目录命名，并加入 GitHub Release 检查更新入口。

## 新增与变更

- 扫描输出目录改为 `扫描方式-扫描步长-月日时分`：
  - 普通扫描示例：`Scan-1um-06251630`
  - 中心扫描示例：`CS-0.5um-06251630`
- 同一分钟重复扫描时自动追加 `-02`、`-03`，避免目录冲突。
- 顶部工具栏新增“检查更新”按钮，自动读取 GitHub 最新 Release；发现新版本后可打开安装包下载页。

## 验证

- `python -m compileall grab_app`
- `python -m pytest tests`

## 安装包

从 GitHub Release 下载：

```text
HTGE采集程序_Setup_v1.1.0.exe
```

安装前建议关闭正在运行的旧版本程序，直接安装新版本即可覆盖更新。

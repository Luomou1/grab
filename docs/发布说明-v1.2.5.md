# v1.2.5 发布说明

`v1.2.5` 增加白色主题，并将检查更新入口收录到设置面板。

## 变更

- 新增设置面板入口，支持在深色 / 白色主题之间切换。
- 主题偏好会保存到本机用户设置，下次启动继续使用上次选择。
- 图标、设备状态、输入框、按钮、Tab、进度条、日志与更新弹窗会随主题切换更新颜色，避免白色主题下文字或图标对比度不足。
- “检查更新”从顶部工具栏移入设置面板。

## 验证

- `python -m compileall grab_app`
- `python -m pytest tests`
- Qt offscreen 主题切换 smoke 检查
- `.\scripts\build_installer.ps1`
